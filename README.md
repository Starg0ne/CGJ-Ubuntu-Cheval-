 Projet réaliser à l'occasion de la Code Game Jam 2026, consistant à créer un Jeu vidéo en seulement 30 heures par équipe. Thème : La fête des clics. Le jeu réalisé est un Platformer.
 Vous incarnez un curseur sur un cheval.
 Parcourez le premier niveau pour atteindre le Festival en un seul morceau. 

 Commandes : 
   - Clic droit : Saut
   - Clic gauche : Déplacement jusqu'à la souris
   - Clic molette + clic gauche : Grapin

Voir : https://astral-panda.itch.io/the-amazing-adventure-of-ubuntu-cheval-click-edition
